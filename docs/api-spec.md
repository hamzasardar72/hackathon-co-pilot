# ClaimWise AI API Specification

## Authentication

- POST `/api/auth/login`
- POST `/api/auth/register`

## Claims

- GET `/api/claims`
- POST `/api/claims/submit`
- POST `/api/claims/analyze`
- PATCH `/api/claims/{claim_id}/review`

## Analytics

- GET `/api/analytics`

## Reports

- GET `/api/reports/download`
